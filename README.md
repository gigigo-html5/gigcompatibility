# gigcompatibility
Module to manage the compatibility of web


## install

`bower install gigcompatibility --save`

## example

```js

    <script>
        var supportSystem = {
            browser: {
                chrome : 27,
                msie : 8,
                firefox: 30,
                safari: 7,
                opera: 8,
                android: true
            },
            system: {
                ios: 6,
                android: 5
            }
        };
    </script>
    
```
